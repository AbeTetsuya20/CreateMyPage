## Getting Started

- Start react application.

```
npx create-react-app <app name>
ex: npx create-react-app mysite

cd <app name>
```

- Clone from git

```
git clone https://github.com/AbeTetsuya20/CreateMyPage.git

cd CreateMyPage
```

- Lower the file hierarchy one level

```
mv * ../

rm -rf CreateMyPage
```

- Install npm module

```
npm install
```

- Start react

```
npm start
```
